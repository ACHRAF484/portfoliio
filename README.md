# Transport & Logistics Professional Portfolio

A clean, elegant, and professional portfolio website showcasing expertise in transport operations, logistics, and supply chain management.

## Features

- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **Professional Layout** - Clean, minimalist design with excellent hierarchy
- **Modern Styling** - Navy blue, grey, and white color scheme
- **Smooth Interactions** - Scroll animations and hover effects
- **Accessibility** - Semantic HTML and clear navigation

## Sections

- **Hero Section** - Professional introduction with call-to-action
- **About Me** - Personal profile and professional summary
- **Experience** - Detailed experience cards with achievements
- **Skills & Tools** - Organized competencies and technical expertise
- **Certifications** - Professional qualifications and education
- **Languages** - Language proficiency with visual indicators
- **Contact** - Easy connection options (Email & LinkedIn)

## Getting Started

### Local Development

1. Clone the repository:
```bash
git clone https://github.com/ACHRAF484/portfoliio.git
cd portfoliio
```

2. Start a local server:
```bash
# Using Python 3
python3 -m http.server 8000

# Using Ruby
ruby -run -ehttpd . -p8000

# Using Node.js
npx http-server
```

3. Open in browser:
```
http://localhost:8000
```

### Deployment

This portfolio can be deployed to:
- **GitHub Pages** - Free hosting directly from your repository
- **Netlify** - Connect your GitHub repo for automatic deployments
- **Vercel** - Simple deployment with preview URLs
- **Traditional Hosting** - Any web hosting service

## Customization

### Update Personal Information

Edit `index.html` to customize:
- Name and title in the hero section
- Professional summary
- Experience details
- Skills and certifications
- Contact information (email and LinkedIn)

### Modify Styling

Edit `style.css` to customize:
- Colors (change CSS variables in `:root`)
- Fonts and typography
- Spacing and layout
- Responsive breakpoints
